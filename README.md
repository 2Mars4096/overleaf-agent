# Skill Overleaf

Prototype Cursor/VS Code extension for browsing, opening, editing, and near-realtime syncing Overleaf project files directly from the editor.

## Status

Planning and discovery.

## Current Direction

- Cursor/VS Code extension, not a Git-sync-only workflow
- Cookie-backed session auth for the MVP
- Text-based project files first; broader asset handling can follow later
- Save-based sync first, then near-realtime refresh/autosync
- Conflict safety over silent overwrite
